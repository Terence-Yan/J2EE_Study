#### 1.Spring装配Bean的方式有哪些？
```
(1).在xml中进行显示配置；
(2).在Java中进行显示配置；
   a.声明配置类：@Configuration
   b.声明Bean: @Bean
   c.装配Bean：@Autowired
   d.使用场景：使用第三方组件，源码不受控制
(3).隐式的bean的发现机制和自动装配
   a.包扫描：@ComponentScan
   b.声明Bean: @Component(@Named)、@Controller、@Service、@Repository
   c.装配Bean：@Autowired
   d.最佳实践：在很多场景下通过组件扫描和自动装配实现Spring的自动化装配是更为推荐的方式。
```
