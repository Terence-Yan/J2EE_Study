#### 1.单例模式1——懒汉式
```
public class SingletonModule1
{
    // 懒汉式
    private static SingletonModule1 instance;

    private SingletonModule1(){}

    public static SingletonModule1 getInstance()
    {
        // 此处存在多线程安全问题
        if(null == instance)
        {
            instance = new SingletonModule1();
        }
        return SingletonModule1.instance;
    }
}
```
